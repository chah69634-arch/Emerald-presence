## 现状（已核对）
`src/windows/activity/components/ActivitySettingsPage.tsx` 有个 `PlaceholderSelect`（`<select disabled>`），以下设置**全是死控件、不生效**：
- 阅读「字体大小」「页面宽度」
- 棋类「棋盘颜色」「棋子样式」
- 「显示调试信息（session_id / FEN）」

（同页「日间/夜间主题」「安全/危险模式」已是真功能，不动。）

## 实现（都走前端本地偏好 `uiPreferences`，无需后端）
用 `getUIPref/setUIPref`（键前缀 `emerald.ui.`，已有）持久化，各活动页读取生效：
1. **阅读字体大小 / 页面宽度**：`activity.reading.fontSize`(如 14/16/18) / `activity.reading.maxWidth`(如 640/760/900)。`ReadingPage` 正文样式读这两个值。
2. **棋盘颜色 / 棋子样式**：`activity.board.theme`(经典木质/冷灰/…) / `activity.chess.pieceStyle`(Unicode 符号/字母)。`ChessPage`/`GomokuPage` 渲染读取。
3. **显示调试信息**：`activity.debug`(bool)。各活动页据此显示/隐藏 session_id / FEN 等原始数据（调试块已有数据，套一个开关即可）。
- 把 `PlaceholderSelect` 换成真 `<select>`/开关，`onChange` 写 uiPref；页面用 `getUIPref` 初始化 + 监听（或重渲染时读）。
- 失败/无值给默认，不报错。

## 验收
- 改字体大小/页宽 → 阅读页即时生效并持久化（重开仍在）。
- 改棋盘配色/棋子样式 → 棋盘渲染变化。
- 开「显示调试信息」→ 活动页出现 session_id/FEN，关则隐藏。
- 不再有 disabled 占位下拉；`npm run build` 通过。
